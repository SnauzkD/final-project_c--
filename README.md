#Поисковой движок (C++)

Учебный проект на C++, реализующий простой поисковый движок на основе
инвертированного индекса.

Проект индексирует набор текстовых документов и позволяет выполнять
поисковые запросы с вычислением относительной релевантности документов.

---

#Возможности

- Инвертированная индексация документов
- Подсчёт частоты слов по документам
- Поиск по запросам из нескольких слов
- Расчёт абсолютной и относительной релевантности
- Сортировка результатов по релевантности
- Покрытие логики unit-тестами (GoogleTest)

---

#Используемые технологии

- **C++20**
- **CMake**
- **GoogleTest**
- **nlohmann/json**
- STL (`map`, `vector`, `algorithm`, `string`)
